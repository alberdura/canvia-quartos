# canvia-quartos

Exercici laravel m7. Pagina de canvi de divises.

## Descripción

`canvia-quartos` es una aplicación web desarrollada en Laravel que permite a los usuarios convertir divisas de todo el mundo. La aplicación proporciona tasas de cambio actualizadas y permite realizar conversiones rápidas y precisas entre diferentes monedas.

## Características

- Conversión de divisas en tiempo real.
- Soporte para múltiples monedas de todo el mundo.
- Interfaz de usuario intuitiva y fácil de usar.
- Actualizaciones automáticas de las tasas de cambio.

## Instalación

1. Clona el repositorio:
    ```sh
    git clone https://github.com/tu-usuario/canvia-quartos.git
    ```
2. Navega al directorio del proyecto:
    ```sh
    cd canvia-quartos
    ```
3. Instala las dependencias:
    ```sh
    composer install
    npm install
    ```
4. Configura el archivo `.env` con tus credenciales de base de datos y otros parámetros necesarios.
5. Genera la clave de la aplicación:
    ```sh
    php artisan key:generate
    ```
6. Ejecuta las migraciones de la base de datos:
    ```sh
    php artisan migrate
    ```

## Uso

1. Inicia el servidor de desarrollo:
    ```sh
    php artisan serve
    ```
2. Abre tu navegador y navega a `http://localhost:8000` para acceder a la aplicación.
